# Counter App
A simple and interactive counter application built using JavaScript. This project demonstrates basic DOM manipulation, event handling, and dynamic UI updates.

## Features

- Increment and Decrement functionality
- Prevents negative values
- Reset button to reset counter
- Dynamic color change:
  -  Green when count >= 10
  -  Black for normal values

## Tech Stack

- HTML
- CSS
- JavaScript

## 📸 How It Works

- Click **+** to increase the count
- Click **-** to decrease the count (won’t go below 0)
- Click **Reset** to reset count to 0
- Color changes dynamically based on value
